#  Verilog RTL Design – Fundamentals (Combinational & Sequential Logic)

This repository contains my implementations of fundamental digital design concepts using Verilog HDL as part of my VLSI learning journey.

The goal of this repository is to build a strong foundation in RTL design by implementing both combinational and sequential logic circuits, 
which are the building blocks of complex digital systems.


##  Contents

###  Combinational Logic
Combinational circuits produce outputs based only on current inputs.

Implemented modules:
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- 2:1 Multiplexer
- 4:1 Multiplexer
- Comparator


###  Sequential Logic
Sequential circuits depend on clock signals and previous states.

Implemented modules:
- D Flip-Flop
- JK Flip-Flop
- T Flip-Flop 
- Shift Registers 


##  Simulation & Verification

Each module includes:
- Verilog RTL design file (`.v`)
- Testbench file for simulation
- Functional verification using simulation waveforms

Tools used:
- EDA PlayGround


## 🎯 Learning Outcomes

Through this repository, I aim to:
- Understand RTL design principles
- Differentiate between combinational and sequential logic
- Gain hands-on experience with Verilog HDL
- Develop skills in writing testbenches and verifying designs
